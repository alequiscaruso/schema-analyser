# srv-engine
GraphSQL and Django Server

# Install
```
$ git clone ...srv-engine
$ cd srv-engine
$ pip install -r requirements.txt
$ cd frontend
$ npm install
$ npm run deploy
$ cd ..
$ python manage.py migrate
$ python manage.py runserver
```
Navigate to http://localhost:8000 and be happy :)
